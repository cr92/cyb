

### Dependencies:
- NodeJS 10.17.0
- MongoDB 4.2.3


### Setting up

#### 1. Start the app
If ports are unavailable, change the params in .env
```sh
$ npm start
```

#### 2. Insert seed data in MongoDB
open another terminal and run
```sh
$ npm run seed
```

#### 3. Run tests
```sh
$ npm run test
```
